Resilient API Client Package
Package Dart autoral desenvolvido com foco em alta disponibilidade e resiliência para o consumo de APIs RESTful.

Arquitetura e Padrões: Implementado utilizando os padrões Strategy e Template Method para centralizar a lógica de repetição (Retry) e garantir o princípio DRY.

Qualidade (TDD): Desenvolvido utilizando Testes Unitários rigorosos com o pacote http/testing, garantindo estabilidade em cenários de falha de rede e timeouts.

Recursos: Suporte a tipos genéricos (Generics) para decodificação automática de modelos, lógica de backoff delay e tratamento de exceções customizadas.


Resilient API Client Package
A custom Dart package designed for high availability and resilience when consuming RESTful APIs.

Architecture & Patterns: Built using Strategy and Template Method design patterns to centralize retry logic and adhere to the DRY principle.

Quality (TDD): Developed with a strict Test-Driven Development approach using http/testing, ensuring robustness against network failures and timeouts.

Features: Provides support for Generics for automated model decoding, exponential backoff delay, and specialized exception handling.
